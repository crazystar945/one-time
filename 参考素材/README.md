# 参考素材

## 用途

本目录存放影片制作过程中用作参考的素材，包括风格参考视频、摄影参考图等。

---

## 目录结构

```
参考素材/
├── README.md
└── 01-参考视频/
```

---

## 01-参考视频

收集同类风格影片片段或镜头参考，用于：

1. **调色参考** — 现实主义工业题材的电影色调
2. **镜头语言参考** — 谈话场景/检修场景的拍法
3. **节奏参考** — 同类廉洁/现实题材短片的叙事节奏

### 建议收集方向

| 类型 | 参考作品方向 | 参考点 |
|------|------------|--------|
| 现实主义工业题材 | 国产电力/工业纪录片及现实题材电影 | 色调、现场收音风格、设备拍摄角度 |
| 廉洁微电影 | 往届"北疆清风颂"获奖作品 | 叙事结构、尺度把握 |
| 谈话场景 | 纪委审查相关影视作品 | 光线设计、空间关系 |
| 师徒/家庭场景 | 国产现实主义家庭题材 | 情感表达方式、表演克制程度 |

## 参考视频应在文件夹内记录来源与参考点

建议在 `01-参考视频/` 内创建一个 `参考清单.md` 文件，记录：

```
| 文件名 | 来源 | 参考点 | 时长 |
|--------|------|--------|------|
| xxx.mp4 | 《xxx》电影片段 | 谈话室光线设计 | 30s |
| xxx.mp4 | 某变电站纪录片 | 设备拍摄视角 | 1min |
```
