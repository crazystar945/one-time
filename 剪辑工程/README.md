# 剪辑工程

## 用途

本目录存放后期剪辑过程中的工程文件、字幕和中间版本输出。

---

## 目录结构

```
剪辑工程/
├── README.md
├── 01-项目文件/       ← 剪辑软件工程源文件
├── 02-字幕文件/       ← SRT/ASS字幕文件
└── 03-输出暂存/       ← 中间版本渲染
```

---

## 01-项目文件

存放剪映、Premiere Pro或其他剪辑软件的工程文件。

**推荐剪辑工具：**

| 工具 | 优点 | 适合场景 |
|------|------|---------|
| **剪映专业版** | AI功能集成（字幕识别/调色），操作简单 | 快速出片，推荐首选 |
| **Premiere Pro** | 专业调色/音视频精细化控制 | 如追求更高画质和精细度 |

**版本管理：**
- 每次正式修改前复制工程文件，另存为新版本
- 命名：`就这一次-V版本号`
- 记录修改日志在工程说明层或标记关键版本

---

## 02-字幕文件

### 字幕内容来源

全部对白和字幕卡依据《06.完整版电影剧本（对白版）.md》抄录。

### 字幕分类

| 类型 | 示例 | 格式建议 |
|------|------|---------|
| 对白字幕 | 调查人员：申报10个，实际用了6个。 | SRT |
| 关键句强调字幕 | "师傅，你教我的规矩还算数吗？" | SRT + 位置居中偏下 |
| 片尾字幕卡 | 没有一个人觉得自己腐败… | 硬字幕（压入视频） |
| 片头标题 | 《就这一次》 | 硬字幕或动态标题 |

### 时间轴标注建议

字幕文件内建议标注场景切换点，方便与视频画面对齐：

```
00:00:00,000 --> 00:01:00,000
（片头：谈话室）
```

## 03-输出暂存

存放未最终定稿的渲染版本。

**输出说明：**
- 每版用版本号标注：`就这一次-V1.0-粗剪.mp4`
- 视频编码：H.264，分辨率 1920×1080
- 保留关键版本（粗剪→精剪→调色版→配音合成版）
- **最终定稿版移入 [成品输出](../成品输出/) 目录**

## 剪辑参考

详细剪辑节奏设计见《12.剪辑节奏脚本.md》，关键参数：

| 时间区间 | 节奏 | 重点 |
|---------|------|------|
| 0:00-4:00 | 缓慢建立 | 长镜头，环境代入 |
| 4:00-9:00 | 逐渐加快 | 蒙太奇，重复动作 |
| 9:00-13:00 | 压迫推进 | 四次反转递进 |
| 13:00-15:00 | 放缓升华 | 重新选择的平静 |

### 调色三阶段

| 阶段 | 色彩 | 对应段落 |
|------|------|---------|
| 信任期 | 暖黄 | 第一幕 |
| 模糊期 | 低饱和/冷蓝灰 | 第二幕+调查 |
| 回归期 | 自然光 | 结尾 |

### 关键转场

| 位置 | 方式 |
|------|------|
| 现实→回忆 | 风声转场+暖色调切入 |
| 蒙太奇段落 | 快速硬切 |
| 调查→证据展示 | 资料匹配剪辑 |
| 结尾（签字） | 笔尖匹配过渡 |
