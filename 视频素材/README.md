# 视频素材

## 用途

本目录存放AI视频生成工具（推荐：**可灵 Kling**）产出的视频片段。这些视频素材是后期剪辑的主要画面来源。

## 生成方法

1. **首帧垫图**：使用[关键帧](../图片素材/关键帧)目录中对应的即梦图片作为第一帧
2. **提示词**：参考《10.AI视频提示词（可灵）.md》
3. **逐段生成**，每段可能需要多次生成选最优

**注意**：
- 如可灵单次生成时长不足，同一段可分段生成后剪辑拼接
- 人物面部若出现不一致，后期可通过画面遮挡（特写/远景）减轻
- 多次生成选最优时，优先保留"表情自然"＞"画面精美"

## 每个视频文件夹内应存放

| 文件 | 说明 |
|------|------|
| `V-XX-生成原片-01.mp4` | AI工具原始输出（保留多版，不要覆盖） |
| `V-XX-选中版.mp4` | 最终选定的最佳版本（供剪辑使用） |
| `V-XX-垫图.jpg` | 生成时使用的关键帧垫图 |

如某段生成了多个候选，**不要删**，标注后在文件夹内保留，选中的复制为"选中版"。
