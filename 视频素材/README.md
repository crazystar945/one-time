# 视频素材总目录

## 当前定位

本目录存放可用于 V2.1 剪辑的所有视频资产。当前以 [《就这一次》AI短片剧本/15.低额度素材生成计划.md](../《就这一次》AI短片剧本/15.低额度素材生成计划.md) 和 [00-项目管理/素材台账.md](../00-项目管理/素材台账.md) 为准。

## 目录职责

- 根目录：存放旧版可复用的视频底片，作为环境镜头或空镜参考。
- [V2-新增](./V2-新增)：只存 V2.1 新生成、已验收并准备入剪的视频素材。

## 目前使用规则

1. 旧版视频只作为环境、反应、回忆或空镜复用，不能直接拼成 V2.1 成片。
2. V2.1 新素材统一进入 [V2-新增](./V2-新增)，命名如 `V2-A2-第一次签字-01.mp4`。
3. 入剪后复制一份为 `-选中版`，原片保留不覆盖。
4. 优先完成 A2—A4 这类证据链核心镜头（A1 为关键帧图片，见图片素材），后续再补 B1、B4、B5 情绪镜头。
5. 旧版“赵建国结尾签字”相关画面不使用；《夜晚小型办公室_李强》仅作 B5 构图参考，不直接入剪。

## 推荐命名

- 原片：`V2-A2-第一次签字-01.mp4`
- 选中版：`V2-A2-第一次签字-选中版.mp4`

## 生成时建议

- 先使用 [图片素材/关键帧](../图片素材/关键帧) 作为首帧垫图。
- 生成时尽量控制时长在 4—5 秒，避免复杂动作和长对白。
- 面部不稳定时优先保留手部、构图和动作清晰的版本。
