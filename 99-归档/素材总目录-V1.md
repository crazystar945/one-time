# 《就这一次》素材总目录

## 目录结构总览

```
廉洁文化/
├── 《就这一次》AI短片剧本/          ← 14份项目文档（剧本/分镜/说明等）
│
├── 图片素材/                       ← 即梦/文生图 产出
│   ├── 人物定型/                  ─ 4个角色的标准形象参考图
│   ├── 关键帧/                    ─ 12张场景关键画面
│   └── 道具定型/                  ─ 8个核心道具/环境参考图
│
├── 视频素材/                       ← 可灵/图生视频 产出（10段）
│   ├── 01-开场调查（0:00-1:00）
│   ├── 02-曾经的榜样（1:00-2:30）
│   ├── 03-第一次人情（2:30-3:30）
│   ├── 04-发现备件异常（3:30-5:00）
│   ├── 05-习惯形成（5:00-8:00）
│   ├── 06-小周质问（8:00-8:30）
│   ├── 07-内心冲突（8:30-9:30）
│   ├── 08-调查与反转（9:30-13:00）
│   ├── 09-最终选择（13:00-14:30）
│   └── 10-结尾升华（14:30-15:00）
│
├── 音频素材/
│   ├── 01-配音/                   ─ 角色台词录音（6角色）
│   ├── 02-配乐/                   ─ 背景音乐（三阶段情绪）
│   └── 03-音效/                   ─ 笔落声/风声/盖章声等
│
├── 剪辑工程/
│   ├── 01-项目文件/               ─ 剪映/Premiere工程源文件
│   ├── 02-字幕文件/               ─ SRT/ASS字幕
│   └── 03-输出暂存/               ─ 中间版本渲染
│
├── 参考素材/
│   └── 01-参考视频/               ─ 风格参考/同类作品
│
├── 成品输出/                       ─ 最终参赛成片
│
└── 素材总目录.md                    ← 本文件
```

## 素材流转链路

```
                  ┌─────────────┐
                  │ 人物定型图   │────→ 用于垫图，保持人物一致性
                  └─────────────┘
                         │
                  ┌─────────────┐
                  │ 道具定型图   │────→ 用于垫图，关键道具一致性
                  └─────────────┘
                         │
                  ┌─────────────┐
                  │ 12张关键帧   │────→ AI视频垫图 + 剪辑视觉参考
                  └─────────────┘
                         │
                  ┌─────────────┐      ┌─────────────┐
                  │ 10段视频素材  │────→│ 配音/配乐/音效 │
                  └─────────────┘      └─────────────┘
                         │                    │
                         └────────┬───────────┘
                                  │
                          ┌───────────────┐
                          │  剪辑合成      │
                          │  (调色+字幕)   │
                          └───────────────┘
                                  │
                          ┌───────────────┐
                          │  成品输出      │
                          │  (参赛成片)    │
                          └───────────────┘
```

## 素材命名规范

| 类型 | 格式 | 示例 |
|------|------|------|
| 人物定型 | `角色名-描述-编号` | `赵建国-正面标准像-01` |
| 关键帧 | `KF-序号-场景描述` | `KF-04-发现备件异常` |
| 道具定型 | `P-序号-道具名` | `P-01-备件清单` |
| 视频素材 | `V-序号-场景描述` | `V-04-发现备件异常` |
| 配音 | `角色-场景序号-台词编号` | `赵建国-S03-01` |
| 配乐 | `BGM-情绪-编号` | `BGM-压迫-01` |
| 音效 | `SFX-音效名` | `SFX-笔落声` |
| 字幕 | `字幕-中文/英文` | `字幕-中文-终稿` |
| 剪辑工程 | `项目-版本号` | `项目-V2.3` |
| 成品 | `就这一次-版本号` | `就这一次-V1.0` |

## 文件备份策略

- 原始素材（图片/视频/音频生成原片）：**不删除**，保留完整
- 剪辑工程文件：每完成一个正式版本另存为新文件，不要覆盖前版
- 成品输出：最终参赛版标明 `V-final`
