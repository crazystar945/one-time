# 成品输出

## 用途

本目录存放最终参赛成片。所有中间版本和工程文件保留在 [剪辑工程](../剪辑工程/) 目录中，这里只放最终定稿版。

---

## 输出规范

| 项目 | 要求 |
|------|------|
| **格式** | H.264 编码 MP4 |
| **分辨率** | 1920×1080 (16:9) |
| **帧率** | 25fps 或 30fps |
| **码率** | 不低于 20Mbps |
| **时长** | 约 15分钟 |
| **字幕** | 硬字幕（压入视频）+ 单独SRT文件备用 |
| **音频** | 立体声 48kHz 320kbps |

---

## 文件命名

```
就这一次-V版本号-日期.mp4
```

示例：
```
就这一次-V1.0-粗剪-0715.mp4
就这一次-V2.0-精剪-0718.mp4
就这一次-V-final-0720.mp4
```

**最终定稿版统一命名为：`就这一次-最终版.mp4`**

---

## 配套材料

成片定稿后，还需要准备以下材料：
- 作品说明（已准备：《13.参赛作品说明.md》）
- 答辩说明（已准备：《14.答辩说明（可选）.md》）
- 剧照（从关键帧或视频中截取）
