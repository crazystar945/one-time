# 成品输出

## 用途

只存放 V2.1 最终送审版成片。中间版本和临时输出放 [剪辑工程/03-输出暂存](../剪辑工程/03-输出暂存/)；旧版 V1 成片见 [99-归档/V1-成品与合成脚本](../99-归档/V1-成品与合成脚本/)。

## 交付技术规格

| 项目 | 要求 |
|------|------|
| **格式** | H.264 编码 MP4 |
| **分辨率** | 1920×1080 (16:9) |
| **帧率** | 25fps 或 30fps |
| **码率** | 不低于 20Mbps |
| **时长** | 12—15 分钟（V2.1 建议值） |
| **字幕** | 硬字幕（压入视频）+ 单独 SRT 文件备用 |
| **音频** | 立体声 48kHz 320kbps |

> 送审前对照 [00-项目管理/廉洁文化主题活动-征集通知.PDF](../00-项目管理/廉洁文化主题活动-征集通知.PDF) 逐条核验格式、时长、署名与 AI 作品说明要求。

