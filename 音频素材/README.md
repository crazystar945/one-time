# 音频素材

> **注意（V2.1）：** 本 README 的场景编号与音效清单属于 V1（含矿泉水等废弃元素）。现行声音设计以 `11.配乐与音效设计.md` 为准（含 27 镜编号与全部 SFX 清单）。

## 用途

本目录存放影片所需的全部音频素材，包括配音、配乐、音效三大类。

## 01-配音

### 角色台词量分布

| 角色 | 台词场景 | 台词量 | 声线要求 |
|------|---------|--------|---------|
| **赵建国** | 全片多场景 | 最多 | 48岁男声，稳重低沉，略带疲劳感 |
| **调查人员** | 场景一、八 | 中等 | 40岁男声，冷静克制，职业感 |
| **小周** | 场景二、四、五、九 | 3-4处关键句 | 26岁男声，清澈干净 |
| **李强** | 场景三、六 | 中等 | 40岁男声，务实疲惫感 |
| **女儿** | 场景七 | 1句 | 10岁女童声 |
| **外委负责人** | 场景九 | 1句 | 普通中年男声 |

### 录音方式建议

| 方式 | 优点 | 缺点 |
|------|------|------|
| **真人录制** | 情感自然，可微调语气 | 需场地设备，需协调人员 |
| **专业TTS（火山引擎/微软Azure）** | 快速，音质干净 | 情感表达有限 |
| **剪映配音** | 最快捷 | 选择有限 |

### 文件夹结构说明

`01-配音/` 下可按角色分文件夹存放录音文件，命名格式：
```
角色-场景序号-台词序号.wav
例：赵建国-S03-01.wav  →  赵建国在场景3的第1句台词
```

台词全文对照剧本《06.完整版电影剧本（对白版）》抄录。

## 02-配乐

### 音乐三阶段

| 阶段 | 对应段落 | 情绪 | 配乐风格 |
|------|---------|------|---------|
| 第一幕 | 场景二~四（信任期） | 温暖 | 轻钢琴+低弦乐，旋律柔和 |
| 第二幕 | 场景五~七（模糊期） | 压迫 | 低频鼓点+环境音，节奏渐紧 |
| 第三幕 | 场景八~九（醒悟期） | 反思→释然 | 单钢琴→宽广弦乐 |

### 音源建议

1. **Suno / AIVA** — AI生成原创配乐，可根据情绪文本生成
2. **素材库**（剪映/EPIDEMIC SOUND等）— 选择现实主义风格的工业/环境配乐
3. **钢琴实录** — 结尾独奏段落建议弹奏实录，更有温度

### 文件命名

```
BGM-情绪-编号.mp3
例：BGM-温暖-01.mp3
     BGM-压迫-01.mp3
     BGM-反思-01.mp3
```

## 03-音效

### 关键音效清单

| 音效 | 场景位置 | 象征意义 | 获取方式 |
|------|---------|---------|---------|
| **笔落声**（三次） | 开场无、中期轻飘、结尾沉稳 | 责任选择 | 实录或素材库 |
| **盖章声** | 蒙太奇段落 | 制度边界 | 素材库 |
| **风声**（三版） | 开头冷、回忆暖、结尾静 | 人物变化 | 素材库 |
| **清单翻动声** | 场景三、八 | 证据感 | 实录翻纸声 |
| **设备运行声** | 检修场景持续背景 | 行业特征 | 素材库 |
| **矿泉水瓶放下声** | 场景二 | 人情入口 | 实录 |
| **脚步声** | 场景一入口 | 人物进入 | 实录 |

### 文件命名

```
SFX-音效名.wav
例：SFX-笔落声-沉稳.wav
     SFX-风声-冬季冷.wav
     SFX-盖章声.wav
```

## 总生产顺序建议

1. **先做配音**（第一批）—— 角色台词是剪辑的时间基准
2. **再做音效**（第二批）—— 关键符号音需要听画面卡点
3. **最后配乐**（第三批）—— 音乐根据最终画面长度和节奏调整
