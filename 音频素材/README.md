# 音频素材总目录

## 当前定位

本目录负责存放最终成片所需的配音、配乐和音效。当前制作顺序以 [《就这一次》AI短片剧本/11.配乐与音效设计.md](../《就这一次》AI短片剧本/11.配乐与音效设计.md) 为准。

## 目录职责

- [01-配音](./01-配音)：角色对白与画外音。
- [02-配乐](./02-配乐)：情绪推进和节奏控制。
- [03-音效](./03-音效)：笔落声、翻页声、环境声、设备声等。

## 建议流程

1. 先用粗剪版本定好节奏与时长。
2. 再录制或生成配音，确保对白与镜头切点一致。
3. 之后补环境音和符号音，最后再配乐。

## 命名建议

- 配音：`角色-场景-序号.wav`
- 配乐：`BGM-情绪-编号.mp3`
- 音效：`SFX-音效名.wav`

## 统一原则

- 声音要服务叙事，避免过多铺陈。
- 符号音如笔落声、翻页声要与画面停顿形成精准呼应。
- 旧版含“矿泉水”等元素的音效设定**不再使用**；音效清单以 `11.配乐与音效设计.md` 第四节为准（含 `SFX-手机震动`）。
