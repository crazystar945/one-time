# 道具定型图片素材

## 用途

本目录存放影片中**重要道具和场景环境**的定型参考图。这些图片用于：
- 确保关键道具在各场景中视觉统一
- 作为AI生成时的垫图，控制画面细节
- 为场景搭建提供参考标准

---

## 素材清单

---

### P-01：备件清单（核心道具）

| 项目 | 内容 |
|------|------|
| 重要性 | ★★★★★ — 贯穿全片的证据核心 |
| 出现场景 | 场景一（开场）、场景三（发现异常）、场景八（调查）、场景九（结尾） |
| 类型 | 纸质表格，B5/A4大小 |

**视觉要求：**
- 标准检修备件消耗清单格式
- 有表头栏（项目名称/备件编号/规格型号/申报数量/实际用量/确认人）
- 关键数字：申报数量栏写"10"，实际用量栏写"6"
- 纸张有折痕、略带磨损（真实感）
- 蓝色或黑色圆珠笔手写痕迹

**生成提示词参考：**

> 电力检修备件消耗清单表格，白纸，B5大小，表格列有项目名称、备件编号、规格型号、申报数量、实际用量、确认人。关键数据行：申报数量=10，实际用量=6，用蓝色圆珠笔手写。纸面有折痕和轻微磨损，放在普通办公桌面上，自然光线，微距特写，真实道具摄影

### P-02：签字笔

| 项目 | 内容 |
|------|------|
| 重要性 | ★★★★★ — 岗位权力的视觉符号 |
| 出现场景 | 全片贯穿，每一次签字都是选择 |
| 类型 | 普通黑色中性笔（如得力/晨光款） |

**视觉要求：**
- 最常见的黑色中性笔，笔身半透明可见墨水
- **不是高档钢笔，不是名牌笔**
- 场景一：放在桌面上，静态
- 场景五：赵建国停在半空的手
- 场景九：笔尖划过纸面的特写

**生成提示词参考：**

> 普通黑色中性签字笔，半透明笔杆可见黑色墨水，放在白色纸张上，微距特写，侧光照明，笔身有低调的品牌logo（得力或晨光款），真实静物摄影，中性色调

### P-03：矿泉水瓶

| 项目 | 内容 |
|------|------|
| 重要性 | ★★★★ — 人情关系的入口符号 |
| 出现场景 | 场景二（检修现场，李强递给赵建国） |
| 类型 | 普通透明矿泉水瓶 |

**视觉要求：**
- 最普通的透明矿泉水瓶，无明显品牌标志或撕掉标签
- 瓶身有水珠（冰镇或天气炎热）
- 已经打开或未开均可
- **不是高级矿泉水，不是功能饮料，不是茶饮**

**生成提示词参考：**

> 一瓶普通透明矿泉水，塑料瓶身，无品牌包装纸或标签已撕掉，瓶身有水珠凝结，放在变电站设备金属台面上，背景虚化有工业设备，午后光线，写实静物，浅景深

| 项目 | 内容 |
|------|------|
| 重要性 | ★★★★ — 制度与流程的视觉载体 |
| 出现场景 | 场景四、场景五、场景九 |
| 类型 | 牛皮纸文件夹或蓝色硬壳文件夹 |

- 内页有检修记录表格

**生成提示词参考：**

> 牛皮纸档案文件夹放在办公桌上，内页有电力检修记录表格，文件夹表面有磨损和使用痕迹，侧面有手写编号标签，自然侧面光，写实摄影

### P-06：审计报告/卷宗

| 项目 | 内容 |
|------|------|
| 重要性 | ★★★ — 调查阶段的证据感 |
| 出现场景 | 场景八（调查室） |
| 类型 | 装订成册的A4报告 |

**视觉要求：**
- 白色封面，装订整洁
- 封面标题格式："内部审计报告"或"专项审计说明"
- 内页有数据表格和时间线
- 堆叠在桌面上，形成"证据量感"

**生成提示词参考：**

> 多份装订好的A4报告堆叠在办公桌上，白色封面印有"内部审计报告"标题，旁边散落几页时间线图表和数据表格，桌面整洁有秩序感，冷色荧光灯光，写实办公场景

### P-07：谈话室环境

| 项目 | 内容 |
|------|------|
| 重要性 | ★★★ — 调查场景的环境定型 |
| 出现场景 | 场景一、场景八 |
| 类型 | 简洁办公房间 |

**视觉要求：**
- 普通基层办公楼房间，不豪华
- 一张桌子，几把椅子（非审讯室风格）
- 墙面浅色，可能有制度牌或宣传画
- 窗户透进自然光，光线冷（冬季）
- **没有强光灯、没有铁窗、没有警察氛围**

**生成提示词参考：**

> 中国北方基层办公楼简洁房间，墙面浅色，一桌几椅，桌面干净只有文件和笔，窗户透进冬季冷光，室内无多余装饰，不豪华不压抑，普通公室环境，中性冷色调，纪实摄影风格

## 生成注意事项

### 一致性要求
- P-01（备件清单）是最重要的道具，在多个场景出现，务必用垫图保持外形一致
- P-02（签字笔）在片头和结尾形成闭环，所用的笔应为同一支
- P-05（工装）与人物定型图中一致

### 负面提示词
- 不要高档文具、不要名牌钢笔
- 不要豪华装修背景
- 不要道具过于崭新（要有使用感）
- 不要出现真实电力企业logo（可用模糊处理）

### 对所有道具的统一要求
真实、朴素、有使用痕迹 —— 这是现实主义的基石。
