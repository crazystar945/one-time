## 关键要求

1. **面部一致性优先**——所有后续生成的关键帧和视频，人物的面部特征应尽量与这里的定型图一致
2. **服装一致性**——每个角色在整部影片中的着装固定（赵建国=深蓝工装，李强=灰蓝工作服，小周=蓝色工装，调查人员=夹克）
3. **不要美化**——这些都是基层劳动者，不要生成时尚杂志风格
4. **不要夸张表情**——角色表情克制、真实，不要大笑或愤怒
