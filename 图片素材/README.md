# 图片素材总目录

## 当前定位

本目录只存放可用于 V2.1 制作的图片资产。优先保证人物一致、道具一致、画面可直接入剪，而不是继续堆叠旧版素材。

## 目录职责

| 文件夹 | 用途 |
|---|---|
| [人物定型](./人物定型) | 角色标准像，后续所有图片与视频都以这里为参考。 |
| [关键帧](./关键帧) | 可复用的关键场景画面，以及 V2 新增关键帧。 |
| [道具定型](./道具定型) | 备件清单、签字笔、记录本、文件夹等核心道具参考。 |

## 目前建议的使用顺序

1. 先用人物定型图定角色形象。
2. 再用道具定型图统一文件、笔、记录本、工装细节。
3. 生成关键帧时优先补 A1、B2、B3、B5 这类图片素材（A2—A4 为视频，见视频素材）。
4. 所有新文件命名统一为 `KF-V2-...` 或 `V2-...`，选中版再复制为 `-选中版`。

## 统一要求

- 画面比例统一为 16:9。
- 风格以写实、纪实、低饱和为主。
- 文字、数字、签名等内容尽量后期叠加，不直接交给 AI 生成。
- 旧版图片仅作参考，V2 生成素材优先进入 [关键帧/V2-新增](./关键帧/V2-新增) 与 [关键帧/镜头脚本](./关键帧/镜头脚本)。
