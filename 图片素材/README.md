# 图片素材

**本目录存放所有图片资产：定型参考图（真实 + AI）+ 关键帧。**

---

## 目录职责

| 文件夹 | 用途 |
|---|---|
| [定型参考](./定型参考) | 角色/道具/环境的真实参考照片与 AI 定型图，统一作为 AI 生成的视觉锚点 |
| [关键帧](./关键帧) | 可复用的关键场景画面，以及 V2 新增关键帧 |

---

## 目前建议的使用顺序

1. 用 `定型参考/角色/` 中的角色定型图锁定人物形象（垫图/参考图）。
2. 用 `定型参考/道具/` 中的道具参考图统一文件、笔、记录本、工装细节。
3. 生成关键帧时优先补 A1、B2、B3、B5 这类图片素材（A2—A4 为视频，见视频素材）。
4. 所有新文件命名统一为 `KF-V2-...` 或 `V2-...`，选中版再复制为 `-选中版`。

---

## 统一要求

- 画面比例统一为 16:9。
- 风格以写实、纪实、低饱和为主。
- 文字、数字、签名等内容尽量后期叠加，不直接交给 AI 生成。
- 旧版图片仅作参考，V2 生成素材优先进入 [关键帧/V2-新增](./关键帧/V2-新增) 与 [关键帧/镜头脚本](./关键帧/镜头脚本)。
