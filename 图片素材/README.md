# 图片素材总目录

## 目录结构

| 文件夹 | 用途 | 数量参考 |
|--------|------|---------|
| [人物定型](./人物定型) | 主要角色的标准形象参考图 | 每个角色1-2张 |
| [关键帧](./关键帧) | 各场景的关键画面（用于AI视频垫图+剪辑参考） | 12-15张 |
| [道具定型](./道具定型) | 核心道具和场景环境的定型参考图 | 6-8张 |

## 生成顺序建议

1. **先做人物定型**（第一批）—— 确保4个角色形象统一，后续所有素材以这些为基准
2. **再做道具定型**（第二批）—— 备件清单等关键道具的形象统一
3. **最后关键帧**（第三批）—— 用定型的人物和道具形象生成各场景画面

## 工具建议

- **即梦**（推荐）：人物写真模式可保持面部一致性
- **Midjourney**：画面质感更好，但人物一致性需垫图控制
- **可灵**：可直接用于后续视频生成的垫图

## 色彩体系对照

生成图片时注意匹配对应段落的色调：

| 段落 | 色调 | 对应场景 |
|------|------|---------|
| 第一幕（信任期） | 暖黄 | 场景二～四 |
| 第二幕（模糊期） | 低饱和/冷蓝灰 | 场景五～七 |
| 第三幕（调查期） | 冷色 | 场景一、八 |
| 结尾（回归期） | 自然光 | 场景九 |
