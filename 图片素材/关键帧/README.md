# 关键帧图片素材

## 用途

本目录存放影片各场景的**关键画面帧**。这些图片用于：
- 作为AI视频生成（可灵）的**首帧垫图**，确保画面与剧本一致
- 后期剪辑时的**画面参考标准**
- 参赛材料中的**剧照示例**

## 生成注意事项

### 人物一致性
- 赵建国在所有关键帧中保持：48岁/深蓝工装/灰短发/肤色偏深
- 李强保持：40岁/灰蓝工作服/偏瘦/疲惫
- 小周保持：26岁/蓝色工装/清爽
- 调查人员保持：40岁/夹克/冷静

### 道具一致性
- 备件清单是核心道具，KF-04和KF-09中务必保持外形一致
- 签字笔在KF-01和KF-11中形成首尾呼应

### 负面提示词（所有图片通用）
- 不要黑帮风格、不要犯罪片风格
- 不要夸张表情、哭喊、愤怒
- 不要现金、红包、银行卡
- 不要豪华办公室、豪车、豪宅
- 不要西服革履
- 不要过度滤镜、不要吸血鬼化妆

### 统一参数
- 画面比例：16:9
- 画质：4K
- 镜头风格：35mm电影镜头，浅景深
- 光线：自然光优先
- 噪点：适当保留胶片质感
