# 关键帧图片素材

## 当前定位

本目录已经不再以“旧版 12 张清单”为唯一依据。当前 V2.1 的关键帧生成以 [《就这一次》AI短片剧本/09.AI分镜提示词（即梦）.md](../../《就这一次》AI短片剧本/09.AI分镜提示词（即梦）.md) 和 [00-项目管理/素材台账.md](../../00-项目管理/素材台账.md) 为准。

## 目录职责

- [V2-新增](./V2-新增)：存放 V2.1 新生成并已验收的关键帧。
- [镜头脚本](./镜头脚本)：按镜头编号整理的定场关键帧，适合直接做垫图或参考图。
- 旧文件：仍可作为历史参考，但不再作为 V2.1 的唯一制作依据。

## 目前建议的使用方式

1. 先用 [图片素材/定型参考/角色](../定型参考/角色) 确定角色形象。
2. 再用 [图片素材/定型参考/道具](../定型参考/道具) 统一道具细节。
3. 生成关键帧时重点补齐 A1、B2、B3、B5 这类图片素材（A2—A4 为视频，见视频素材）。
4. 新文件命名优先使用 `KF-V2-...`，选中版再复制为 `-选中版`。

## 统一要求

- 画面保持写实、纪实、低饱和的风格。
- 人物服装、工装颜色、年龄设定须与人物定型保持一致。
- 文字、数字、签名尽量后期叠加，避免 AI 误生成。
- 旧版场景“矿泉水”“赵结尾签字”**不使用、不入剪**（台账 L01/L03）；“李强夜间办公室”旧图仅作 B5 构图参考。
