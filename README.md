# 《就这一次》AI短片项目

## 从这里开始

本项目当前以 **剧本 V2** 为唯一创作依据。制作时按以下顺序进入文件：

1. [项目控制台](./00-项目管理/README.md) — 当前版本、制作顺序、交付前检查。
2. [完整版电影剧本（V2）](./《就这一次》AI短片剧本/06.完整版电影剧本（对白版）.md) — 唯一剧情与对白依据。
3. [镜头脚本（V2）](./《就这一次》AI短片剧本/08.镜头脚本（ShotList）.md) — 画面、声音与剪辑依据。
4. [低额度素材生成计划](./《就这一次》AI短片剧本/15.低额度素材生成计划.md) — 每日生成优先级。
5. [素材台账](./00-项目管理/素材台账.md) — 生成、验收、入剪状态。

## 目录职责

```text
00-项目管理/                 项目控制台、资产台账、版本决策
《就这一次》AI短片剧本/       创作源文件（06、08、15 为 V2 当前依据）
图片素材/人物定型/            锁定的人物参考，不重复生成
图片素材/关键帧/              旧版可复用关键帧
图片素材/关键帧/V2-新增/      仅存 V2 新生成、验收通过的关键帧
视频素材/                     旧版可复用原始视频，绝不覆盖
视频素材/V2-新增/            V2 新镜头的原片与选中版
音频素材/                     配音、配乐、音效
剪辑工程/                     工程文件、字幕、临时输出
成品输出/                     当前交付版；历史版本放入历史版本/
参考素材/                     外部风格参考，不进入正式剪辑素材池
```

## 三条硬规则

1. **不覆盖原始文件。** 新生成文件只进 `V2-新增`；入选后复制为 `-选中版`。
2. **不以旧提示词为制作依据。** `09.AI分镜提示词（即梦）.md` 和 `10.AI视频提示词（可灵）.md` 尚未同步 V2，仅作旧版存档参考。
3. **一个事实只能有一套道具。** “实际 6、确认 10”的文件、赵建国签名、结尾记录本“6”，统一由后期制作并在所有镜头复用。
