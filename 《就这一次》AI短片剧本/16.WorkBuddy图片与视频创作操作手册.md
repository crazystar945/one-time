# 《就这一次》WorkBuddy 图片与视频创作操作手册

**对应版本：V2｜目标：用最少生成次数补齐 A1—A4 核心素材。**  
**先读：** [低额度素材生成计划](./15.低额度素材生成计划.md)、[V2 镜头脚本](./08.镜头脚本（ShotList）.md)。

> WorkBuddy 的页面字段名称可能略有不同。无论它叫“文生图 / 图生视频 / 参考图 / 首帧 / 提示词”，都按本手册的顺序与内容填写即可。

## 一、总原则：先锁人，再锁道具，最后做动作

### 必须准备的参考图

在 WorkBuddy 新建项目时，先上传本地已有的角色定型图：

| 角色 | 本地文件 |
|---|---|
| 赵建国 | `图片素材/人物定型/赵建国-正面标准像-01.png` |
| 李强 | `图片素材/人物定型/李强-正面标准像-01.png` |
| 小周 | `图片素材/人物定型/小周-正面标准像-01.png` |
| 调查人员 | `图片素材/人物定型/调查人员-正面标准像-01.png` |

**使用规则：**

1. 三人同框场景上传赵建国、小周、李强三张参考图；谈话室上传赵建国、调查人员两张。
2. 能设置“参考强度/人物一致性”时，选中高档或 70%—85%；太高会让动作僵硬，可降到 65% 后重试。
3. 全部生成横版 16:9。视频单条 4—5 秒即可。
4. 人物不要说话、不要正面口型；所有对白后期配音。
5. **文件、手机、记录本不要让 AI 写字。** 数字“6”“10”、赵建国签名、手机消息均在剪映/PPT/PS 后期叠加。

## 三、第一步：生成 A1 关键帧

### A1 的用途

这张图要反复用于：第一次发现异常、第一次签字首帧、调查材料、文件局部特写。它是全片最重要的一张图。

### WorkBuddy 操作

1. 选择“文生图”或“参考图生图”。
2. 上传赵建国、李强、小周的定型图。
3. 选择 16:9 横版、写实电影感、最高可用清晰度。
4. 粘贴下列“正向提示词”和“负面提示词”。
5. 一次生成 1—2 张即可，最多尝试 4 次；选“桌面最完整、三人关系最清楚”的一张。

### A1 正向提示词（完整复制）

```text
现实主义廉洁主题微电影关键帧，16:9横版，中国北方变电站黄昏时分的临时办公室，真实基层电力检修工作环境，工业纪录片电影质感，35mm镜头，台灯暖黄光照亮桌面，窗外是收工后模糊的钢铁设备和傍晚冷色天空。

人物严格参考上传的人物定型图：
赵建国，48岁中国男性，肤色偏深，短发微灰，穿旧而整洁的深蓝色电力检修工装，坐在桌前，是画面中心；
李强，40岁中国男性，偏瘦有风霜感，灰蓝色工作服，站在门边，紧张但不奸诈；
小周，26岁中国男性，整洁蓝色工装，站在桌侧，认真看向文件。

三人形成自然三角构图。赵建国右手手指停在桌面两张空白确认文件上方，黑色签字笔放在文件边。桌面有文件夹、检修记录本、普通水杯。两张A4文件必须完整、平整、留有大面积纯白无文字区域，方便后期叠加“实际6个、确认10个、签名”等内容。人物表情克制，空气安静紧张，不要戏剧化哭喊。高质量真实电影剧照，自然皮肤纹理，低饱和，背景适度虚化。
```

### A1 负面提示词（完整复制）

```text
不要任何可读文字，不要数字，不要签名，不要乱码，不要水印，不要品牌logo，不要现金红包银行卡，不要警察手铐审讯灯，不要西装领带，不要豪华办公室，不要反派脸，不要夸张愤怒或哭喊，不要矿泉水递送动作，不要夜间小企业办公室，不要过度磨皮，不要卡通风格。
```

### A1 验收与保存

- 选中标准：赵居中；三人外貌一致；桌面完整；纸张足够干净；没有错误文字。
- 保存为：`图片素材/关键帧/V2-新增/KF-V2-A1-6与10-选中版.png`
- 在 [素材台账](../00-项目管理/素材台账.md) 将 A1 改为“已选中”。

## 四、第二步：生成 A2“第一次犹豫后签字”视频

### WorkBuddy 操作

1. 选择“图生视频”。
2. 首帧上传 A1 选中版；如可多参考图，再加入赵建国和小周定型图。
3. 选择 16:9、4—5 秒、固定镜头或轻微推进。
4. 不输入对白，不要求中文口型。

### A2 视频提示词（完整复制）

```text
基于首帧画面生成4到5秒现实主义电影镜头。黄昏变电站临时办公室，固定桌面近景。赵建国穿深蓝电力检修工装，右手握黑色签字笔，在空白确认单上方先犹豫停顿约两秒，然后缓慢、沉重地落笔签字。镜头重点是“停顿后落笔”的完整动作，纸面保持无文字无数字。小周穿蓝工装站在桌侧后景，轻微虚焦，安静看着，不说话，不走动。李强可在门边虚焦静止。台灯暖黄光照在手和纸上，窗外昏暗，真实中国基层工作环境，克制、安静、无戏剧化表演，镜头不晃动。
```

### A2 负面提示词

```text
不要可读文字，不要数字，不要签名特写文字，不要中文口型，不要人物说话，不要手部变形，不要多余手指，不要快速镜头运动，不要夸张表情，不要现金红包，不要水印，不要字幕。
```

### A2 验收

- 必须看到：停顿 → 落笔；手与笔稳定。
- 人脸即使略有变化也可用，后期优先裁切手部。
- 保存为：`视频素材/V2-新增/V2-A2-第一次签字-选中版.mp4`

## 六、第四步：生成 A4“小周写下真实数字 6”视频

### WorkBuddy 操作

1. 可上传现有“一年后的变电站检修现场”图片或视频首帧作环境参考。
2. 上传小周、赵建国定型图。
3. 重点要求：小周是主体，赵建国只在后景整理工具；不要让赵再次签字。

### A4 视频提示词（完整复制）

```text
4到5秒现实主义电影镜头，一年后的夏日中国北方变电站检修现场，自然明亮日光，钢铁设备和远处铁塔真实清晰。小周，26岁中国男性，穿整洁蓝色电力工装，已经是现场负责人，在夹板上的空白记录本认真写下一行内容。镜头以小周的手、记录本和专注侧脸为主体。赵建国，48岁，穿普通深蓝检修工装，胸前没有班长标识，在后景整理工具箱，停下后平静点头；赵建国不接文件、不签字。现场秩序安静，设备稳定运行，缓慢轻推镜头，真实工业纪录片质感，克制、有重量感，人物不说话、不做口型，记录本无任何可读文字。
```

### A4 负面提示词

```text
不要可读文字，不要直接生成数字6，不要乱码，不要赵建国签字，不要赵建国穿班长标识，不要外委人员抢画面，不要口型，不要夸张英雄姿势，不要夕阳史诗感，不要水印，不要手部变形。
```

### 后期关键动作

在剪映/PPT/PS 将记录本书写区域遮罩替换为清晰的 **“6”**。这是 V2 结尾的唯一正确数字画面。

保存为：`视频素材/V2-新增/V2-A4-小周写6-选中版.mp4`

## 八、生成完后的入库与剪辑

1. 原始候选不要覆盖，统一加编号 `-01`、`-02`。
2. 最佳版本另复制一份，加 `-选中版`。
3. 图片存入 `图片素材/关键帧/V2-新增/`；视频存入 `视频素材/V2-新增/`。
4. 在 `00-项目管理/素材台账.md` 依次修改状态：待生成 → 已生成待验收 → 已选中 → 已入剪。
5. A1—A4 完成后立即做 V2 粗剪；不要等到所有 B 级素材都生成完。

## 九、常见问题

| 问题 | 处理方式 |
|---|---|
| 文件上出现乱码或错误数字 | 不修 AI 文字；用后期文件模板遮挡替换。 |
| 人脸不像定型图 | 降低人物动作要求，改拍侧脸、手部、背影；视频中优先裁切。 |
| 手指/笔变形 | 重生成一次；仍不稳就只取前后 1 秒，配静帧和笔尖声。 |
| 画面过于“反腐剧” | 添加“克制、现实主义、无审讯感、无犯罪片灯光”；去掉强烈冷光词。 |
| 每日额度不够 | 当天只做 A1 或 A2，第二天继续；绝不跳过 A1 直接生成复杂多人物视频。 |
