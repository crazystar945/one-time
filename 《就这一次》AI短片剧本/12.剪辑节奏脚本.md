# 《就这一次》剪辑节奏脚本

**对应版本：V2.1｜建议总时长：12—15 分钟｜剪辑原则：证据递进，不做“反转”字幕。**

## 一、剪辑主线

```text
谈话室的“6 / 10”问题
→ 曾经严谨的师徒日常（埋入李强队伍的困境）
→ 第一次知道6却签10（问工资、看设备后才落笔）
→ 无记录却写“已现场核实”
→ 不看数据便签字，小周质问
→ 李强夜里发出“老办法”试探
→ 家庭中的“诚实”与“老办法”
→ 三张签字单还原事实，差额翻了十倍
→ 调离审核岗
→ 小周写下真实的6
```

任何不服务这条线的旧镜头（矿泉水、赵建国结尾签字）不进入时间线；李强夜间独处按 V2.1 压缩版（B5，镜头14）重拍，旧素材只作参考。

## 二、时长与节奏

| 段落 | 建议时长 | 节奏 | 剪辑目标 |
|---|---:|---|---|
| 谈话室冷开场 | 0:00—0:45 | 慢 | 让观众先看见 6 与 10。 |
| 师徒现场 | 0:45—2:00 | 平稳 | 建立赵建国的专业与小周的信任。 |
| 第一次签字 | 2:00—4:20 | 放慢 | 让“停顿→落笔”被看清。 |
| 第二次确认 / 质问 | 4:20—6:30 | 逐渐加快 | 从例外转为习惯。 |
| 李强试探、家庭与短信 | 6:30—7:45 | 突然放慢 | 李强发出“老办法”；家庭中“诚实”形成内心回声。 |
| 调查材料 | 7:45—10:30 | 冷静推进 | 三张单据证明责任链。 |
| 醒悟与处理 | 10:30—11:30 | 留白 | 承认越线、交代岗位后果。 |
| 一年后 | 11:30—13:30 | 平稳 | 小周按实记录，赵支持但不替代。 |
| 片尾 | 13:30—15:00 | 收束 | 设备与风声，字幕落点。 |

> 若素材量不足，可压缩到 10—12 分钟；不要为凑 15 分钟重复空镜或延长无意义对白。

## 三、关键段落剪法

### 1. 冷开场：先给事实，后给解释

1. 冬日变电站远景 3—5 秒。
2. 谈话室中景，赵建国与调查人员 5—7 秒。
3. 拆检记录“6”与确认单“10”局部特写 3 秒（后期文件叠字）。
4. 赵建国回答“我知道”，切风声进入夏日回忆。

不要展示银行流水、抓捕或快切悬疑包装。

### 2. 第一次签字：全片第一个不可跳过的动作

镜头顺序：

1. 桌面完整文件与赵的手指。
2. 李强解释，切窗外收工人群与送电确认声。
3. 小周一句“六个就是六个”。
4. 笔被放到文件旁。
5. 赵低声问“工资几个月没发齐了”，视线扫过修得规矩的设备——体谅在这里成形。
6. 赵停顿；A2 只保留“停顿→落笔”最可信的 3—4 秒。
7. 切小周无声反应。

**不要剪成“赵只是放下清单”。** V2 的责任必须由签字动作成立。

### 3. 第二次确认与小周质问

将 A3 的快速签字与资料室电话振动、文件退回单交叉。小周的质问前先留一拍赵没有看数据的动作；质问说完后至少留 2 秒设备低鸣和赵回避，不要立即配乐或解释。

### 4. 家庭段

家庭段前先给 B5：李强夜里未接“工头”来电、发出信息、“已读”无回复（镜头14），不进音乐。家庭段内，女儿作文纸和赵写“诚实”的手，优先于正反打对白。手机“老办法”作为一个 1—2 秒插入镜头（镜头16），与 B5 使用同一套聊天界面道具；信息内容后期叠加。赵扣下手机后直接切谈话室桌面，形成“私人沉默→公共责任”的匹配剪辑。

### 5. 调查段

不做“第一反转、第二反转”标题卡。按材料顺序推进：

1. 赵说“我没收钱”。
2. 调查人员说明没有收钱、也没有送钱。
3. 三张单据并列：6签10、无记录写已核实、后来未核对。
4. 审计比对表点出数字：第一次四个接触头，两年账面差额翻了十倍。
5. 调查人员指出确认权的责任。
6. 笔被推回赵面前，赵承认第一次已经越线。

每次材料出现都使用同一套文件版式；不要生成或展示错误的 AI 文字。

### 6. 尾声

先给“调离原审核岗位、依规整改”的黑场字幕，再进入一年后。外委提出“先写进去”时，赵不接文件；镜头主体始终是小周写记录本。最后特写必须是**小周写下 6**，而不是赵建国签字。

## 四、低额度素材拼接方法

| 缺口 | 用法 |
|---|---|
| 谈话室反应 | 用现有谈话室视频裁切赵、调查人员、桌面三个景别。 |
| 6 / 10 文件 | A1 静帧裁为全景、数字特写、手指特写三次使用。 |
| 调查材料 | B3 静帧做轻微推拉，叠加文件翻页声与配音。 |
| 家庭段 | B2 静帧缓推、手机画面后期合成，不强求口型视频。 |
| 一年后 | A4 + 现有变电站全景 + 工具箱背影，完成完整段落。 |

## 五、粗剪验收

- [ ] 没看剧本的观众能说出“实际 6、确认 10、赵签了”。
- [ ] 观众能分清第一次签字与第二次“已现场核实”。
- [ ] 调查段不用任何“反转”文字仍能听懂责任在哪里。
- [ ] 结尾由小周写 6；赵不签字、不恢复原审核岗位。
- [ ] 所有字幕、文件、配音与 V2 对齐。
