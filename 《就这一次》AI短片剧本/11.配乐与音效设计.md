# 《就这一次》配乐与音效设计

**对应版本：V2｜原则：环境声先于音乐，证据声先于煽情。**

## 一、声音母题

| 母题 | 出现 | 含义与处理 |
|---|---|---|
| 笔尖划纸 | 第一次签字、第二次签字 | 第一次清晰、停顿后落笔；第二次更快、更轻。不是“漂亮的签字声”，而是责任落纸。 |
| 笔被推回 | 调查谈话 | 调查人员把笔推向赵建国；只留桌面摩擦声，不让赵再次写字。 |
| 记录本书写 | 尾声 | 小周写下真实数字“6”；声音平稳、清楚，完成价值闭环。 |
| 风声 | 开场 / 回忆 / 结尾 | 冬风冷；夏风轻；结尾风声退去，只剩设备稳定运行。 |
| 设备低鸣 | 所有现场 | 始终存在，提醒“设备修好”不等于“手续正确”。 |
| 文件翻页 | 黄昏办公室 / 谈话室 | 事实一层层显现，避免用悬疑鼓点制造刺激。 |

## 二、分段声音设计

### 1. 谈话室与回忆切入

- 开场：冬风、纸张翻动、杯子与桌面极轻接触声；不进音乐。
- 调查人员提出“实际几个”后，留 1—2 秒静默，再用风声切入夏日现场。

### 2. 师徒检修与第一次签字

- 检修现场：工具、脚步、远处设备低鸣，配乐只用极轻的低弦或不用。
- 黄昏办公室：送电确认声淡出后，音乐完全停下。
- 李强说“就这一次”后，不加煽情低音；保留呼吸、纸张、台灯电流声。
- 赵建国落笔：笔尖划纸声放大，但不做夸张音效；随后直接切小周反应。

### 3. 第二次确认与小周质问

- 资料室：电话振动、打印机远声、翻页声；赵快速落笔时笔声短促。
- 小周质问：音乐归零，只有设备低鸣。赵转身离开时，不加音乐回应。

### 4. 家庭与“老办法”信息

- 李强板房独处（镜头14）：只用手机震动、板房外风声和椅子细响；信息发送后留 1 秒静默盯着“已读”，不进音乐。
- 家庭开始可有极轻的室内环境与暖调钢琴单音。
- 女儿说“说错了也要改过来”后，钢琴停。
- 手机亮起只用轻微提示音；“老办法”文字由后期叠加，不需要语音读出。

### 5. 调查与醒悟

- 调查段以纸张、笔在桌面滑动、椅子细响推进；不使用银行流水、键盘或审讯音效。
- 三张签字单并列时，可叠加极轻的纸张落桌声。
- “这支笔不收钱，也会留下责任”后停 2 秒，只留赵建国呼吸。
- 赵承认第一次越线后，不加哭戏或高潮音乐；用单音钢琴衔接走廊脚步。

### 6. 尾声

- 字幕“调离原审核岗位”时无音乐。
- 一年后：自然风、远处工具声、稳定设备低鸣。
- 小周写“6”时记录本书写声最清楚；赵说“照着做”后才进入极轻的钢琴与弦乐。
- 最终远景只保留风声和设备声，片尾字幕前 2 秒淡出音乐。

## 三、配乐清单（少而精）

| ID | 用途 | 建议时长 | 说明 |
|---|---|---:|---|
| BGM-01-检修日常 | 师徒现场 | 30—45 秒 | 温和低弦/极简钢琴，不抒情。 |
| BGM-02-习惯形成 | 第二次确认前后 | 30—60 秒 | 低频环境纹理，不能像犯罪悬疑片。 |
| BGM-03-看清 | 调查后至走廊 | 20—35 秒 | 单音钢琴，留大量空白。 |
| BGM-04-尾声 | 小周写6后 | 25—40 秒 | 轻钢琴与极淡弦乐，不做胜利感。 |

## 四、必须准备的音效文件

`SFX-冬风`、`SFX-夏风`、`SFX-设备低鸣`、`SFX-翻文件`、`SFX-笔第一次签字`、`SFX-笔快速签字`、`SFX-笔桌面滑动`、`SFX-记录本书写`、`SFX-走廊脚步`、`SFX-手机轻提示`、`SFX-手机震动`。

所有文件存入 `音频素材/03-音效/`。配音先于配乐录制，配乐只能在粗剪锁时长后铺设。
