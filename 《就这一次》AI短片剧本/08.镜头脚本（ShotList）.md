# 《就这一次》镜头脚本（Shot List）

**对应剧本：V2.1｜建议时长：12—15分钟｜风格：克制、纪实、少配乐**

> 执行原则：不把人物拍成"坏人"，也不替错误辩护。每一次越界必须有明确、可见、可追溯的书面动作；每一次沉默都保留反应镜头。

---

## 一、视觉与声音母题

| 母题 | 叙事功能 | 使用方式 |
|---|---|---|
| 黑色签字笔 | 岗位确认权与责任 | 开场桌面、第一次签字、调查桌面；结尾不再由赵建国签字 |
| "6 / 10" | 事实与被改写的结算依据 | 拆检记录写"6"，确认单写"10"；同一份道具在调查段重现 |
| 记录本 | 师徒传承与制度延续 | 小周开场记下规程，结尾写下真实数字"6" |
| 风声与设备低鸣 | 环境压力与秩序 | 风声切入回忆；关键沉默只留设备声；结尾回归平稳运行声 |
| 门/走廊 | 边界与后果 | 谈话室门、资料室门、家门、处理后的走廊，避免重复特写 |
| 手机屏幕 | 关系惯性与试探 | 李强夜间发出"老办法"信息，赵在家中收到；"已读"无回复即是回答 |

色彩路径：**谈话室冷蓝灰 → 现场暖而不艳 → 越界后低饱和 → 一年后自然日光。** 不使用犯罪片式强对比光或夸张推拉。

### 镜头 1｜场 1：谈话室｜⏱ 18s

| 维度 | 内容 |
|---|---|
| **景别/运动** | 远景，固定机位（脚架），无任何运动。广角镜头（等效 24—28mm），大景深（f/8—f/11）。 |
| **灯光/色调** | 自然光，清晨/冬日冷蓝（色温 ~5600K）。天空佔画面 2/3，地面设备佔 1/3。低反差，无暖色。 |
| **构图/焦点** | 三分法构图。钢架和龙门架剪影在下 1/3，铁塔中景排列成引导线，远景天空泛冷白。焦平面覆盖全景（无限远）。 |
| **画面描述** | 冬日变电站全景。前景钢架和龙门架剪影，中景铁塔沿画面纵深排列，导线在风中轻微晃动。地面有薄霜或干裂泥土。天空微亮、冷白色，无云或薄云。无人物、无烟雾、无冷却塔、无烟囱。 |
| **表演/走位** | 无人物。 |
| **声音设计** | 持续低频风声（40—60Hz 范围），远处变压器稳定嗡鸣（50Hz 基频），偶有绝缘子轻微放电"嘶嘶"声。约 18 秒。 |
| **对白** | 无。 |
| **关键道具** | 无。 |
| **衔接** | 入：黑场渐入（~1.5s）。出：硬切至镜头 2。 |

### 镜头 2｜场 1：谈话室｜⏱ 18s

| 维度 | 内容 |
|---|---|
| **景别/运动** | 中景（腰部以上），缓推至近景（胸部以上）。稳定器/轨道，推进速度极慢、几乎不易察觉。中焦镜头（等效 35—50mm）。 |
| **灯光/色调** | 窗户为唯一光源，阴天柔光，色温 ~5500K。赵脸部半明半暗（侧光），照度低。无补光。整体冷蓝灰色调。 |

| **画面描述** | 谈话室。赵建国（48 岁）坐在桌前，工装洗得干净但袖口有磨旧毛边。他盯着桌面——桌上有一支黑色签字笔、一摞复印件、一杯没动过的水（水面平、无波纹）。窗外的冬光冷而均匀，在他脸上打出半面柔和阴影。他不说话也不动。 |
| **表演/走位** | 赵：静止坐着，眼睛盯笔，喉结偶有极轻微的上下移动。无大幅度表情。呼吸平稳。 |
| **声音设计** | 画外翻纸页声（调查人员手在画外翻文件，约 2—3 次翻页）。室内空气感——安静但不舒适，有轻微室内混响。 |
| **对白** | 无（调查人员画外音在镜头 3）。 |
| **关键道具** | 黑色签字笔（居中前景、笔帽未盖）、一摞复印件（右前侧，约 5—8 张）、水杯（透明玻璃杯、水满的、没动过、左前侧）、窗户（背景唯一光源，无窗帘或有百叶窗但全开）。 |
| **衔接** | 入：硬切自镜头 1。出：硬切至镜头 3。 |

### 镜头 3｜场 1：谈话室｜⏱ 22s

| 维度 | 内容 |
|---|---|
| **景别/运动** | 文件特写（90° 俯拍）→ 缓拉至赵近景。俯拍时焦距 ~50mm（防畸变）；拉起后等同 ~35mm。运动平滑，约 8—10 秒完成拉起。 |
| **灯光/色调** | 同镜头 2 冷蓝灰。桌面照度因文件白色反光略高于人脸。文件上的"6"和"10"需要有轻微阴影（侧面柔光）。 |
| **构图/焦点** | 俯拍：拆检记录（左）与确认单（右）各佔画面一半，对角线排列。拉起后：赵在画面上方 1/3（胸以上），文件在下方 2/3。焦点从纸面→转移至赵的眼睛。 |
| **画面描述** | 两张纸并排——左：现场拆检记录，手写"6 个"（铅笔/蓝色墨水）；右：《备件消耗确认单》，打印"10"，右下角有赵建国亲笔签名。调查人员的手指（白衬衫袖口，画外进入）从右侧伸入，指腹点在签名旁，不盖住签名。赵建国抬起头，喉结动了一下（吞咽）。切至：确认单上"赵建国"签名——极近特写，墨迹在纸纤维上轻微洇开的质感可见。 |
| **表演/走位** | 调查人员：只有右手入画（白衬衫袖口，40 岁左右男性的手指，指甲干净），指签名。赵：低头看纸→抬头（速度不快，像在拖时间）→看向调查人员→吞咽→嘴微张（想说什么但最终只说"我知道"）。 |
| **声音设计** | 调查人员（画外、平静、不咄咄逼人）："这张确认单上，是十个。你的字。" 赵（轻声）："我知道。"——停顿 2 秒。签名特写时，风声渐起（低音量切入，预示回忆段落）。 |
| **对白** | 调查人员："这张确认单上，是十个。你的字。" 赵建国："我知道。" |
| **关键道具** | 拆检记录（A4，手写"6 个"，铅笔或蓝墨水，笔迹工整）、确认单（A4 打印表格，"10"是打印体，签名"赵建国"为黑色签字笔亲笔，同支笔）、调查人员的手（白衬衫袖口）。 |
| **衔接** | 入：硬切自镜头 2。出：风声渐起转场至镜头 4。 |

### 镜头 4｜场 2：变电站检修现场｜⏱ 28s

| 维度 | 内容 |
|---|---|
| **景别/运动** | 中景，手持跟拍（轻微晃动，纪实感，模拟在场目击者的视角）。跟随赵建国的步伐平移。中焦（等效 35mm）。 |
| **灯光/色调** | 夏日午后自然日光，色温 ~5000K，光线偏硬（阳光直射）。高光有轻微过曝（设备金属表面反光）。整体暖而不艳，饱和度正常。与镜头 1—3 的冷蓝灰形成鲜明对比。 |
| **构图/焦点** | 赵居中，小周在后侧跟（画左后方）。控制柜在前景佔左下角。焦点在赵的手指和编号铭牌上。深景深（f/5.6—f/8），现场环境清晰。 |
| **画面描述** | 烈日下变电站。钢架、绝缘子串和运行中的设备发出嗡鸣。赵建国蹲在控制柜前，手指一排编号铭牌逐项核对。小周（26 岁，干净工装）拿着记录本站旁边，一边听一边记。汗水在赵的鬓角——不是汗流浃背，是正常夏日作业的汗。背景远处有零散的其他检修工人。 |
| **表演/走位** | 赵：蹲→手指顺着铭牌号码一一滑过→说话时眼看柜子不抬头→说完"别图快"时偏头扫了小周一眼→继续核对下一组。小周：站着→点头→低头记→偶尔抬眼确认编号。赵起身时膝盖轻微"咔"一声（自然细节，不强调）。 |
| **声音设计** | 设备嗡鸣（持续低频 ~100Hz + 50Hz 谐波），远处金属工具偶尔碰撞声（清脆，约 3—4 次）。对白：赵语速正常，语调平坦（是重复过很多次的规程提示，不是说教）。 |
| **对白** | 赵建国："拆一个，记一个；换一个，销一个。别图快。" 小周："记着呢。" 赵建国："现场能返工，记录不能返工。写下去的，就是责任。" |
| **关键道具** | 设备编号铭牌（金属、银色蚀刻）、小周记录本（牛皮纸封面，约 B5 大小，翻开可见铅笔字迹）、赵鬓角的汗（化妆）。 |
| **衔接** | 入：风声转场（从镜头 3 延续，风声音高上升后切至设备声）。出：硬切至镜头 5。 |

### 镜头 5｜场 2：变电站检修现场｜⏱ 12s

| 维度 | 内容 |
|---|---|
| **景别/运动** | 特写，固定机位。长焦微距（等效 85—100mm）拍记录本；标准中焦拍扳手动作。两个微切（或一镜完成）。 |
| **灯光/色调** | 同镜头 4 夏日午后日光。记录本纸面有阳光反射但不刺眼。 |
| **构图/焦点** | 记录本特写：纸面佔满画幅，字迹在中央。扳手动作：画面可见赵的手和小臂（画右进入），扳手在画面中下位置。浅景深（f/2.8—f/4），焦点在字迹/扳手接触面。 |

## 四、素材生成与剪辑要点

1. **道具连续性是硬要求。** "6"和"10"的笔迹、纸张版式、赵建国签名必须在镜头 3、7、9、17、19 中完全一致；建议先制作一套可复用的主道具高清图。

2. **赵建国的表演曲线：** 场 4 眼神干净、动作有序；场 8 低声问工资、看设备后才下决心；场 9 犹豫后主动签字；场 11 回避；场 12 烦躁；场 21 不是崩溃而是终于看清；场 26 平静、不抢小周的主体性。

3. **李强只拍"困难"，不拍"可怜"。** 汗湿工装、不断来电、等结算的焦虑足够；镜头 14 的独处段落用"未接来电+输入又删掉+已读无回复"三个动作完成，不能用哭诉、病人、欠薪群像去替违规辩护。

4. **调查段只用材料与停顿推进。** 不加"第几次反转"字幕，不加审讯式拍桌和激烈音乐。调查人员始终平静，力量来自证据链。

5. **签字声只用三次。** 场 9 最清晰且刺耳；场 11 更短更快；场 21 只见笔被推回、不再落笔。结尾改为小周写"6"，完成价值反转。

6. **手机信息的收发闭环：** 镜头 14（李强发出）与镜头 16（赵收到）使用同一条信息内容、同一聊天界面道具；"已读"标记在 14 中给特写。

7. **节奏建议：** 镜头纯时长约 617 秒（场 1—9 约 200 秒；场 10—16 约 169 秒；场 17—23 约 152 秒；场 24—27 约 96 秒），加上场间转场/黑场呼吸约 1.5—2 分钟、片尾字幕约 40 秒，成片总长预计 12—13 分钟。
