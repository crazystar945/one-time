# 《就这一次》低额度素材生成计划

**目标：** 在免费额度有限的前提下，补齐新版剧本 V2 的叙事硬点；优先保证“事实清楚、人物一致、可剪辑”，不追求每个镜头都由 AI 视频生成。  
**执行原则：** 先复用已有素材，后生成不可替代镜头；一张关键帧尽量服务多个镜头；文件文字、数字和字幕全部后期制作，不交给 AI 生成。

---

## 0. 开始前：先不要生成这些

- **不要再生成人物定型图。** 现有赵建国、李强、小周、调查人员定型图已经足够，后续一律作为垫图/参考图使用。
- **不要生成带可读文字的文件。** AI 常把“6、10、已现场核实、老办法”等字生成错误。确认单、拆检记录、手机消息、审计表都应在剪映、PS、Canva 或 PPT 中后期叠字。
- **不要生成长对白视频。** AI 口型与人物一致性不稳定。人物对话用 3—5 秒无声/环境声动作镜头，加后期配音和反应镜头完成。
- **不要沿用旧版“矿泉水”“赵建国结尾签字”“李强夜间办公室”的素材设定。** 它们已不在 V2 叙事链中，不值得消耗新额度。

## 3. 推荐生成顺序（按免费额度分日）

### 第 1 天：先锁住人物与道具（图片日）

1. 生成 **A1“6/10桌面三人构图”**，挑一张作为整片文件场景的统一底图。
2. 生成 **B3“调查桌面俯拍”**，构图和 A1 的文件尺寸、笔颜色保持一致。
3. 若仍有额度，生成 **B2“家庭作文”**。
4. 立即在本地制作一套统一文件道具：拆检记录（6）、确认单（10）、情况说明、记录本（6）。

### 第 2 天：只生成最不可替代的动作（视频日）

1. 生成 **A2 第一次犹豫后签字**。
2. 生成 **A3 第二次快速签字**。
3. 生成 **A4 小周写下“6”**。
4. 每条只留最佳一版；人物脸不稳时，优先选手部清楚、构图稳定的版本。

### 第 3 天：补情绪，不补解释

1. 生成 **B1 小周质问 / 赵离开**。
2. 生成 **B4 走廊背影**。
3. 额度不足时，B1、B4 都可由关键帧做 105%→112% 的缓慢推拉，配后期声音完成。

## 4. 低额度剪辑替代方案

| 需要表现的内容 | 不必新生成的视频方案 |
|---|---|
| 调查人员逐条指出问题 | 现有谈话室视频 + 文件局部静帧 + 调查人员配音 + 赵建国反应镜头。 |
| “6”与“10”的对照 | 用 A1 静帧裁成三个不同景别：全桌面、数字特写、赵的手指；观众会以为是三个镜头。 |
| 赵建国犹豫 | A2 取首帧停 1 秒，再接落笔动作；加放大的呼吸与笔尖声。 |
| 时间流逝、习惯形成 | A2 / A3 两段签字镜头交叉，配不同日期的文件叠层与电话声；无需生成“第四次签字”。 |
| 女儿提问 | 家庭静帧缓推 + 女儿画外音 + 赵建国手停住的裁切特写；不必追求口型。 |
| 一年后正确选择 | A4 小周写“6” + 赵建国后景点头 + 现有变电站全景；比让赵再次签字更符合新版主题。 |

## 5. 每次生成前的固定设置

1. 使用对应人物定型图作参考；同一角色的工装颜色、发型、年龄不得变化。
2. 图片和视频统一 **16:9、自然光、35mm纪实电影感、低饱和、无可读文字**。
3. 视频时长以 **4—5 秒** 为宜；不要要求复杂走位、多人同时说话或完整中文口型。
4. 每个提示词都附加负面词：`夸张表情，哭喊，现金，红包，警察抓捕，豪华办公室，西装，犯罪片灯光，乱码文字，水印`。
5. 文件类素材先留白：后期统一叠加同一套文件和数字，保证 Shot 3、7、9、16、18 的证据连续。

---

## 6. 达标线与停止线

**最低可剪版本：** 已有素材 + A1、A2、A3、A4，共 **1 张关键帧 + 3 条短视频**。这四项完成即可开始正式剪辑。  
**推荐完成版本：** 再加 B1、B2、B3、B4，共 **3 张关键帧 + 5—7 条短视频**。  
**停止线：** A 级已完成且粗剪能讲清“6→10→签字→审计→小周写6”，就停止补景。把剩余额度留给修复人物一致性或重做最差的一条，而不是继续生成空镜。

## 7. 生成后命名

保存到相应素材目录，永不覆盖原片：

- `图片素材/关键帧/KF-V2-A1-6与10-01.png`
- `视频素材/V2-A2-第一次签字-01.mp4`
- `视频素材/V2-A3-快速签字-01.mp4`
- `视频素材/V2-A4-小周写6-01.mp4`

选中的文件另复制一份并命名为 `-选中版`。这样后续剪辑不会误用旧版“矿泉水”或“赵建国结尾签字”的素材。
